# Scrimba interview project
AI-generated visual explanations with narration, rendered live in your browser.

## Pre-requirements
- Required env variables:
```
S3_ENDPOINT="https://xxxx.supabase.co/storage/v1/s3"
S3_ACCESS_KEY="your_access_key"
S3_SECRET_KEY="your_secret_key"
S3_BUCKET="your-bucket-name"
S3_REGION="us-east-1"
OPENAI_API_KEY="openai_api_key"
ELEVENLABS_API_KEY="eleven_labs_api_key"
```
- Node.js/Bun pre-installed


## Getting Started
```bash
npm i
```


## Development
To start the development server run:
```bash
npm run dev
```

Open [localhost:3001](http://localhost:3001/) in your browser.

Live demo: [scrimba.mariothedev.com](https://scrimba.mariothedev.com/) in your browser.
